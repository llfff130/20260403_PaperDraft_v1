# 20260403_PaperDraft_v1
This is the supplementary material for SmartDyGNN: Dynamic Sparse Training for GNNs via Structural and Gradient Fusion, including detailed ablation analysis, training dynamics, hyperparameter sensitivity, generalization results on GAT, full algorithm pseudocode, and extended discussions on limitations and future directions; it validates that fusing structural cues, dynamic embedding signals, and gradient guidance enables adaptive node selection, reduces computational cost by 25%–30% with negligible accuracy loss, and supports plug‑and‑play integration with mainstream GNN architectures such as GCN and GAT.
